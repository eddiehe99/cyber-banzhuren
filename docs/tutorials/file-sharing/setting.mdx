# 配置

## 设置文件共享

:::note

可参考 [在 Windows 中通过网络共享文件](https://support.microsoft.com/zh-cn/windows/experience/connectivity-networking/file-sharing-over-a-network-in-windows)。

:::

## 访问共享文件夹

1. 打开“文件资源管理器”或者“我的电脑”
2. 在地址栏访问 `\\教师电脑的 IP 地址`（如：`\\10.185.241.2`）
3. 输入用户名和密码
    - 优先：用户名 `Everyone`，密码为空
    - 备用：用户名 `Guest`，密码为空
4. 点选“记住我的凭据”，避免每次访问都要输入用户名和密码
5. 将该访问路径添加到“快速访问”或“桌面”，方便下次访问
