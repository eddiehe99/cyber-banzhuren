# 介绍

Banzhuren Notifier 附加、可选的功能。

## 功能

班主任将通知记录在一个通知 Word 文档里，家长将留言记录在 [飞书文档](https://docs.feishu.cn) 中，配合 [自动传递班主任通知](../../bat)，此工具能实现：

- 在班主任电脑上，自动抓取家长留言到班主任的通知 Word 文档里。
- 在教室电脑上，在课间自动打开通知 Word 文档（里面有家长留言）。
- 在教室电脑上，在上课前自动关闭通知 Word 文档（里面有家长留言）。
