# 介绍

本工具可部分实现 [希沃云班牌](https://www.seewo.com/article/detail/396) 的功能。

## 功能

班主任将通知记录在一个通知 Word 文档里，此工具能实现：

- 在教室电脑上，在课间自动打开班主任通知 Word 文档。
- 在教室电脑上，在上课前自动关闭班主任通知 Word 文档。
