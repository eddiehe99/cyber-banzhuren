## 已知问题

- OEM 显卡 Radeon 625 无法安装公版驱动，需要手动安装 AMD Radeon Software
  - 参考 [[显卡] [已解决, 附教程] amd的驱动面板去哪里下载?](https://www.chiphell.com/thread-2747964-1-1.html)

- 使用 WLAN 执行高数据量吞吐任务时，容易造成死机。建议使用有线网络。
- 使用 USB Type-C 充电时，容易对同侧的 USB Type-A 接口连接的无线设备形成干扰。建议使用另一侧的 USB 接口。

## 可扩展
