# 配置（cpzx）

1. 在浏览器访问 [172.16.30.2](http://172.16.30.2/homepage/index.html) 并登录
2. 查看本机的 IP（如：`10.185.241.2`）及 物理（MAC）地址（如：`00:11:22:33:44:55`）
    - 可通过 `ipconfig /all` 命令查看，也可通过设置中的“网络和 Internet”查看
3. 返回浏览器的 [172.16.30.2](http://172.16.30.2/homepage/index.html) 页面，并点击“添加设备”
    - 输入本机的 IP 和 MAC 地址
4. 重新连接网络，并**编辑 IP 分配**
5. 选择**手动分配**，并开启 IPv4
6. 输入网络信息
    - IP 地址：本机的 IP（如：`10.185.241.2`）
    - 子网掩码：`255.255.255.0`
    - 默认网关：本机 IP 的最后一个分段改成 1（如：`10.185.241.1`）
    - 首选 DNS：`119.29.29.29`
    - 备用 DNS：`223.5.5.5`
