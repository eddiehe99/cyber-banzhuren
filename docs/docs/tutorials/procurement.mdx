## 验收

```mermaid
graph LR;
    filling(教师填写验收表)-->signature(三位教师签名);
    signature-->reimbursement(去财务处走剩余报销流程);
```

:::danger

表格中的项目与数量，一定要与发票相符。

:::

验收表填写示例：

- 填写：日期、表格内容
- 留空：其他内容（批复编号、申报部门、金额单位等）

| 序号 | 采购项目名称 | 品牌 | 数量 |
| :---: | :--- | :--- | :---: |
| 1 | 水火箭尾翼 | 七波 | 8 |
| 2 | 电工胶带 | 七波 | 8 |

- 验收人：**三位**老师签名

:::info 【验收表】由【审批表】修改而来

- 抬头：“申报审批表”改成“申报验收表”；
- 删除表格三列：“规格及技术参数”、“预算单价”、“预算金额”；
- 删除表格整行：采购金额合计（大写）：___ 元；小写：¥ __ . __；
- 修改:“申请人”改成 **“验收人”**，三位老师签名；
- 删除表格内容：**“申报部门负责人：”**、**“联系电话：”**；
- 删除表格单元格：**“校长办公室审批意见：”** 所在单元格

:::

## 报销

```mermaid
sequenceDiagram
    participant 财务 as Liu Weiling （西教学楼一楼财务室）
    participant 教师 as 教师
    participant 领导 as 领导

    教师->>财务: 1. 带申报表、验收表、发票
    财务-->>教师: 2. 发放“支出凭单”
    教师->>教师: 3. 填写凭单
    教师->>教师: 4. 签名(经手人)
    教师->>领导: 5. 提交凭单请领导签字
    领导->>教师: 6. 签名(“证明人”、“领导批准”)
    教师->>财务: 7. 将全部材料交给财务
```
